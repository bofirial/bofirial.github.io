Featured Projects

  • Dungeon Bot

    I am currently working on a game where the player can program robot characters using C# in a browser. The robots can be used to complete quests and defeat enemies in dungeons.

    Dungeon Bot is a work in progress.

    The project is built in Blazor taking advantage of Roslyn and Monaco for client-side code editing.

  • Blazor Character Generator

    I created this web application to compare the performance of client-side Blazor to server-side Blazor / Razor Components.

    The application is based on a horoscope. It accepts some simple input then hashes it and uses the hash to display random stats for a "Technology Character" that could star in an IT based video game.

  • Blazor Four In A Row

    I created this web application to learn more about Blazor, Azure Functions, SignalR, CosmosDB and Azure.

    The application allows any number of players to play a game of "Four In A Row" in real time by dropping pieces into a grid. Teams are selected randomly and there is a delay for each player after dropping a piece.

  • Google Tasks Synchronizer

    I created this Azure Function to synchronize Google Tasks between multiple google accounts. Google does not provide a way to share tasks so this runs frequently to compare tasks across all accounts and send updates to keep everything synchronized.

  • Vitality Squares Probability

    I created this web application to learn Angular 2. It calculates the likelihood of winning a minigame from an incentives based application CareSource uses. It also lets you play the minigame.
