Featured Projects

  • Blazor Technology Character Generator

    This application compares client-side Blazor to server-side Blazor / Razor Components. The same application is displayed in 2 iframes. There is also a form that controls both applications and performance statistics.

    The application itself is based on a horoscope. It accepts some simple input then hashes it and uses the hash to display random stats for a "Technology Character" that could star in an IT based video game.

  • Blazor Four In A Row

    I designed this application to learn more about Blazor, Serverless Architecture, SignalR, NoSQL databases and Azure.

    The application allows any number of players to play a game of "Four In A Row" in real time by dropping pieces into a grid. Teams are selected randomly and there is a delay for each player after dropping a piece.

  • Google Tasks Synchronizer

    This is an Azure Function that I wrote to synchronize google tasks between multiple google accounts. Google does not currently provide a way to share tasks so this runs frequently to make sure that when I make any change to a task that that change is also made on my wife's account and vice-versa.

  • Vitality Squares Probability

    This is a site I built to learn Angular 2. It calculates the likelihood of winning a minigame from an incentives based application CareSource uses. It also lets you play the minigame.
